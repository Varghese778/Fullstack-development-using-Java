## INSTALLATION STEPS

1. Install JDK 17+
   Verify: java -version

2. Install Maven 3.6.0+
   Verify: mvn -version

3. Install Git
   Verify: git --version

4. Clone Repository
   git clone <repository-url>
   cd MSP2-FSD

5. Install Dependencies (Maven will handle automatically)
   mvn clean install

6. Import into IDE
   Eclipse: File -> Import -> Existing Maven Projects
   IntelliJ: File -> Open -> Select MSP2-FSD folder

7. Run Application
   Right-click project -> Run As -> Spring Boot App
   OR: mvn spring-boot:run

## DATABASE SETUP

### Development (H2 - No Setup Needed)
Default configuration uses H2 in-memory database
H2 Console: http://localhost:8080/h2-console
JDBC URL: jdbc:h2:mem:testdb
Username: sa
Password: (empty)

### Production (MySQL)
1. Create database:
   mysql -u root -p
   CREATE DATABASE job_portal;
   CREATE USER 'jobportal'@'localhost' IDENTIFIED BY 'SecurePassword@123';
   GRANT ALL PRIVILEGES ON job_portal.* TO 'jobportal'@'localhost';
   FLUSH PRIVILEGES;

2. Configure: src/main/resources/application-prod.properties
   spring.datasource.url=jdbc:mysql://localhost:3306/job_portal
   spring.datasource.username=jobportal
   spring.datasource.password=SecurePassword@123

3. Run with MySQL:
   mvn spring-boot:run -Dspring-boot.run.arguments="--spring.profiles.active=prod"

## TROUBLESHOOTING

Port 8080 in use:
  Windows: netstat -ano | findstr :8080 && taskkill /PID <PID> /F
  Linux/Mac: lsof -ti:8080 | xargs kill -9
  Or change port: server.port=8081 in application.properties

Maven not finding dependencies:
  rm -rf ~/.m2/repository
  mvn clean install -U

H2 connection failed:
  Ensure spring.h2.console.enabled=true in application.properties

Compilation errors:
  Right-click project -> Maven -> Update Project
  Or: mvn clean rebuild
