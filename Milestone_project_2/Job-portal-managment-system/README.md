# Job Portal Management System - Documentation

## Project Overview

A comprehensive web application connecting job seekers and employers with features for profile management, resume uploads, job listings, applications, and employer dashboards.

### System Architecture
- **Backend:** Spring Boot + Spring MVC
- **Database:** MySQL / H2
- **Frontend:** HTML5/CSS3, Thymeleaf, JavaScript (ES6)
- **Security:** Spring Security with Role-Based Access Control

### Technology Stack
- Spring Boot (REST APIs, Controllers)
- Spring Data JPA (Database Access)
- Spring Security (Authentication & Authorization)
- Thymeleaf (Server-side Templating)
- MySQL/H2 (Database)
- Bootstrap/CSS3 (Responsive UI)
